2025.08.21 - DeepSeek-V3.1 发布

DeepSeek-V3.1 主要变化：

  • 混合推理架构：一个模型同时支持思考模式与非思考模式；
  • 更高的思考效率：相比 DeepSeek-R1-0528，DeepSeek-V3.1-Think 能在更短时间内给出答案；
  • 更强的 Agent 能力：通过 Post-Training 优化，新模型在工具使用与智能体任务中的表现有较大提升

DeepSeek API 也已同步升级，deepseek-chat 对应非思考模式，deepseek-reasoner 对应思考模式，且上下文均已扩展为 128K。同时，API Beta 接口支持了 strict 模式的 Function Calling，以确保输出的 Function 满足 schema 定义。详见 Function Calling
