欢迎来到我的技术博客！这里记录了我在软件开发过程中的学习心得、技术探索和项目经验。

## 关于博客

这个博客涵盖了多个技术领域，从软件架构设计到前端开发，从容器化技术到人工智能应用。每篇文章都是我在实际工作中的思考和总结，希望能对读者有所帮助。

## 文章分类

- **人工智能**: Transformer、RAG、LLMs、Vibe Coding 等
- **.NET 开发**: ASP.NET Core、配置系统、日志、管道等
- **前端开发**: Vue.js、React、TypeScript、Next.js 等
- **容器化与微服务**: Docker、Docker Swarm、微服务挑战与解决方案
- **架构与设计模式**: CQRS、事件驱动架构、中介者模式等
- **领域驱动设计**: DDD 概念、战略设计、战术设计
- **智能家居**: 树莓派、Home Assistant、家庭服务器搭建
- **Linux 运维**: 系统管理、文件操作、安全配置等
